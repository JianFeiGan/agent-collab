## 技术亮点

### 1. 插件系统
```python
# 通过 entry_points 注册自定义 Agent
[project.entry-points."agent_collab.agents"]
my-agent = "my_package.agents:MyAgent"
```

### 2. 条件执行引擎
```yaml
tasks:
  - id: check
    agent: reviewer
    prompt: 检查代码质量
  - id: optimize
    agent: developer
    when: "{{ check.score < 80 }}"
    prompt: 优化代码
```

### 3. 分布式任务队列
```python
queue = InMemoryTaskQueue()
await queue.enqueue(task)
worker = await queue.dequeue()
```
